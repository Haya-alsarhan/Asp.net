PROJECT MONITORING Web-based Application

project Monitoring Web-based Application includes adding files for specific project ,so user can upload files to be shared , download or deleted. adding new users, deleting user, adding new project and specifying manager for the project , selecting members of the group and asking them for meeting and sending an email notification to them. The responses of approve or reject a delivery are sent back to the project manager. and the client can send a messages to the project manager with questions about a specific project. Actions performed are controlled by the manager.
